# 🌱 Eco-Health Assistant

A specialized AI personal assistant expert in nutrition, fitness, and healthy lifestyle, built with Python and Streamlit.

## 🚀 Key Features
* **Expanded Expertise:** Provides expert advice on nutrition, workout routines (gym), physical recovery, and sleep optimization.
* **Contextual Memory:** Uses advanced session state management to retain conversation history for a natural dialogue.
* **Strict Domain Guardrails:** The AI is strictly bounded to health and fitness topics, politely declining unrelated queries (e.g., coding, history, automotive).
* **Safety First:** Implements an automated medical disclaimer in every response to ensure responsible AI usage.
* **Empathetic Interaction:** Configured with specific system instructions to maintain a supportive and friendly tone in Romanian.

## 🛠️ Tech Stack
* **Frontend/Backend:** Python, Streamlit
* **AI Engine:** Google Gemini (via the latest `google-genai` SDK)
* **Environment:** Secure API key management using `python-dotenv`

## 💡 Use Case
This project demonstrates a production-ready implementation of a "Niche AI Agent." It showcases how to use Prompt Engineering to create a safe, focused, and helpful assistant that knows its limits and prioritizes user well-being.
